# Getting Started with dbc GUI

The dbc GUI is a desktop application for managing ADBC drivers.

## Installation

Download the latest release from the [releases page](https://github.com/columnar-tech/dbc/releases) for your platform:

- **macOS**: `.dmg` file (Apple Silicon or Intel)
- **Windows**: `.msi` installer
- **Linux**: `.AppImage` or `.deb` package

## First Run

1. Launch the app
2. The Catalog page shows available drivers from the public registry
3. Click a driver card to see details and install it
4. Installed drivers appear on the Installed page

## Authentication

For private registry access, go to the **Auth** page and log in with:
- **Device Code** (OAuth): Opens your browser for one-click approval
- **API Key**: Enter your key directly
