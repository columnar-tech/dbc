# GUI Workflows

## Browse and Install a Driver

1. Open the **Catalog** page
2. Search for a driver by name
3. Click a card to open the details drawer
4. Select a version and click **Install**
5. Watch real-time progress in the install modal

## Manage a Project Driver List

1. Open the **Project** page
2. Click **Open Project** and select your project directory
3. Add drivers with the search input
4. Click **Sync** to install all listed drivers

## Uninstall a Driver

1. Open the **Installed** page
2. Find the driver and click **Uninstall**
3. Confirm the dialog

## Authenticate with a Registry

1. Open the **Auth** page
2. Enter the registry URL
3. Choose **Device Code** or **API Key** tab
4. Follow the prompts
